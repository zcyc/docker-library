# Coze on Docker  

This is an **[Docker Compose template](https://github.com/Websoft9/docker-library)** powered by [Websoft9](https://www.websoft9.com) based on Docker for Coze:


 - community:  latest


## System Requirements

The following are the minimal [recommended requirements](https://www.coze.cn/):

* **RAM**: 4 GB or more
* **CPU**: 2 cores or higher
* **Disk**: at least 8 GB of free space
* **bandwidth**: more fluent experience over 100M
